Portfólio Acadêmico

Projeto para registro contínuo da minha evolução acadêmica

Sobre este Projeto

Este projeto foi desenvolvido com o objetivo de documentar e organizar minha evolução ao longo do curso de Desenvolvimento de Sistemas.

O site funciona como um diário técnico de estudos, onde cada disciplina possui sua própria página com registros das aulas, conceitos aprendidos, exercícios e exemplos práticos de código.

As anotações são organizadas em formato de cards interativos, renderizados dinamicamente com JavaScript, permitindo visualizar rapidamente a data da aula e expandir o conteúdo completo quando necessário.

O projeto também inclui imagens e capturas de código utilizadas durante as aulas, que podem ser ampliadas para melhor visualização.

Todo o projeto é versionado com Git e publicado através do GitHub Pages, permitindo acesso público ao progresso do desenvolvimento.


Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Git
  • GitHub Pages

Funcionalidades

  • Registro de aulas organizado por disciplina
  • Renderização dinâmica de conteúdo com JavaScript
  • Cards expansíveis para leitura das anotações
  • Suporte para múltiplas imagens em cada aula
  • Visualização ampliada de imagens
