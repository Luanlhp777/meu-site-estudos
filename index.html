Sobre este Projeto

Este portfólio acadêmico está sendo desenvolvido com o objetivo de documentar, organizar e aplicar na prática os conteúdos aprendidos ao longo do curso de Desenvolvimento de Sistemas.

Cada disciplina possui sua própria seção dedicada, onde são registrados conceitos, exercícios, práticas e evoluções técnicas. O projeto também funciona como laboratório pessoal de desenvolvimento web, permitindo aplicar continuamente HTML, CSS e JavaScript em um ambiente real.

Mais do que um repositório de anotações, este site representa minha evolução técnica, disciplina e comprometimento com a área de tecnologia.
